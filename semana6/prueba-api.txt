Petición POST a http://localhost:3000/registro con el JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Respuesta que devuelve el servidor:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Explicación: El servidor responde con un objeto JSON que confirma la recepción
de los datos y devuelve los mismos campos `nombre` y `mensaje`.
